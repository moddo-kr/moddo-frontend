## 📂 디렉토리 구조

```
src/
├── assets/          # 정적 리소스 (이미지, 아이콘 등)
├── common/          # 공용 모듈
│   ├── components/  # 공용 UI 컴포넌트
│   ├── constants/   # 상수 정의
│   ├── hooks/       # 커스텀 훅
│   ├── loaders/     # 라우트 데이터 로더
│   ├── queries/     # tanstack query 관련 로직
│   ├── types/       # 타입 정의
│   └── utils/       # 유틸 함수
├── mocks/           # MSW 핸들러
├── pages/           # 라우트 단위 화면
├── service/         # API 호출 로직
├── stories/         # Storybook 스토리 문서
└── styles/          # 테마, 글로벌 스타일
```
